# Sistema de Administração de Ambientes

Sistema completo de gerenciamento de ambientes, permissões e sincronização de arquivos.

## Características

- Gerenciamento de ambientes de produção
- Sistema de permissões por perfis
- Integração com múltiplos bancos (V8, Presença Bank, HubCredito)
- Sincronização de arquivos entre ambientes
- API REST completa
- Interface web moderna

## Instalação

```bash
npm install
npm start
```

## Configuração

O sistema usa arquivos JSON para armazenamento de dados e variáveis de ambiente para configuração.

## Testes

O projeto inclui configuração para TestSprite MCP para testes automatizados.
